# 核心生成式 AI 技術

在這堂課中，你將學習如何實際運用生成式 AI 技術來構建 AI 驅動的 .NET 應用程式。內容包括大型語言模型的補全與聊天功能、檢索增強生成（RAG）、音頻/視頻分析，甚至是 AI 代理。
